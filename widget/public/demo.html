Getting Started

Home Assistant can be installed on a Raspberry Pi, an old laptop, a NAS, or a dedicated appliance. The recommended installation method is Home Assistant OS which provides a managed environment with automatic updates.

Tip: Use the chat widget in the bottom-right corner to ask questions about integrations, automations, or troubleshoot issues. It's powered by an AI assistant trained on resolved GitHub issues.

Once installed, navigate to http://homeassistant.local:8123 in your browser. The onboarding wizard walks you through creating your first user account and discovering devices already on your network.

Configuration

Most configuration can be done via the UI, but advanced users can edit configuration.yaml directly. The configuration file lives in the /config directory and supports splitting into multiple files using !include directives.

Validate your configuration before restarting with the Developer Tools → Check configuration button, or run ha core check from the CLI.

Common Issues

If a device stops responding, check the integration logs under Settings → System → Logs. For Z-Wave or Zigbee devices, verify the USB stick is recognized and the correct path is set in the integration config.

For database issues, consider adjusting the recorder purge settings or migrating to an external database such as MariaDB or PostgreSQL for larger installations.
